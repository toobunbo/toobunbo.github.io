**Tag** — a mono metadata chip in vermilion tint. Use for writeup techniques and categories.

```jsx
<Tag>pwn</Tag>
<Tag>deserialization</Tag>
<Tag as="a" href="#" active>web</Tag>
```

Leading `#` by default (`hash={false}` to drop it). `active` fills vermilion for a selected filter. Render `as="a"` or `as="button"` to make it clickable.
