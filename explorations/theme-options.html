font pairing × color — 3 options
2026.07 — issue #00

Three directions where type and color travel as one set — because they have to tell the same story. All start from the same base: cream paper, warm black ink (never #000), with dark mode as a calculated negative print. The differences are in the texture of the reading type (Mincho serif "printed book" vs Gothic sans "modern blog") and in how each one answers the red-seal vs terminal question.

RED SEAL vs TERMINAL — on a light paper ground quick verdict

朱 — Vermilion seal ink

On cream, vermilion sits like real stamped ink: warm, weighty, high-contrast, unmistakably print. Bonds naturally with the 易/難 difficulty seals.

→ PRIMARY ACCENT

墨緑 — Terminal green (deep)

Only holds on paper when dark enough; still reads faintly "digital". Not a brand color, but excellent for code / tags / terminal — a disciplined secondary role.

→ SECONDARY (code only)

Cyan — Neon hacker

Goes washed-out on cream, like faded highlighter, and drags in glowing-screen associations → breaks the printed-paper illusion. Cut it.

→ NOT USED

活版 Why A tells the story RECOMMENDED DEFAULT

Mincho serif next to a Reggae One masthead feels like a technical book with a manga cover: the long-form reading has an absolute "printed page" quality while the headlines still shout like a Shonen magazine. A single Japanese family (Shippori) covers both body and the 連載 / 番外編 labels → weight and x-height line up, no clash.

Vermilion is the brand ink; the 易/難 seals use exactly this color. This is the safest option for readability and the most faithful to "a club drawing its own manga." Trade-off: a serif body reads a touch classical for people used to modern tech blogs.

劇画 Why B tells the story

Rampart One is a 3D relief manga logo — the punchiest wordmark of the three, ideal for a landing/dashboard that performs loud. The body, Zen Kaku Gothic New, is a clean humanist sans with a tall x-height that reads very easily on screen for a modern tech blog; the same family also covers the Japanese labels.

This is the "modern blog on paper" route: personality concentrates in the wordmark and seals, the reading stays neutral. Space Mono adds a bit of charming terminal texture. Trade-off: Rampart is very loud — it needs the right dose, or it starts to look like event graphics.

電脳 Why C tells the story

C is A plus one disciplined color rule that answers the brief directly. Vermilion = brand ink (UI, seals, headings). Terminal green = exists only inside code / terminal blocks — so the "hacker" flavor is present but never slides into neon cyberpunk, because it's framed like printed code on a manga page.

It's at its best when dark mode is imagined as "opening a terminal" — that's when the phosphor green is released. Trade-off: the "green stays in the box" rule must be enforced strictly, or the color system dilutes.
